# UsageBar

UsageBar is a native macOS menu bar app for tracking coding plan usage across Bailian, Z.ai Global, and OpenAI Codex.

## Features

- Native SwiftUI menu bar app with a light frosted-glass dashboard
- Bailian usage via web session capture
- Z.ai subscription and quota window monitoring
- OpenAI Codex usage via local Codex OAuth and CLI fallbacks
- Chinese and English UI switch
- Auto refresh with per-provider cached snapshots

## Run

Open [UsageBar.xcodeproj](/Users/spicyclaw/MyProjects/UsageBar/UsageBar.xcodeproj) in Xcode and run the `UsageBar` scheme on `My Mac`.
